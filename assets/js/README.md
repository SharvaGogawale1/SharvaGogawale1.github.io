# JavaScript Directory

This directory is for any JavaScript files you might need to add interactivity to your website.

## Current Setup

The website is currently static HTML with minimal JavaScript needs.

## Potential Use Cases

You might add JavaScript here for:
- Mobile menu toggle functionality
- Smooth scrolling enhancements
- Contact form handling
- Analytics integration
- Interactive research project demos
- Dynamic content loading

## Example Files You Might Add

- `main.js` - General site functionality
- `mobile-menu.js` - Mobile navigation toggle
- `smooth-scroll.js` - Enhanced smooth scrolling
- `contact-form.js` - Contact form validation and submission
- `analytics.js` - Google Analytics or other tracking

## Best Practices

- Keep JavaScript minimal for fast loading
- Use modern ES6+ syntax
- Add proper error handling
- Consider performance impact
- Test across different browsers

## Including JavaScript

Add JavaScript files to your HTML before the closing `</body>` tag:

```html
<script src="./assets/js/main.js"></script>
```
