# Documents Directory

This directory contains document assets for your website, primarily PDFs and other downloadable files.

## Required Files

- **cv.pdf** - Your academic CV or resume

## Document Guidelines

### CV/Resume (cv.pdf)
- **Format**: PDF
- **Content**: Academic CV with your research, publications, education, and experience
- **File Size**: Keep under 2MB for faster downloads
- **Naming**: Use `cv.pdf` (lowercase) to match the HTML link

## Additional Documents

You can add other documents here such as:
- Research papers and preprints
- Project presentations
- Awards and certificates
- Recommendation letters (if sharing publicly)

## File Organization Tips

- Use descriptive filenames with hyphens (e.g., `research-paper-2024.pdf`)
- Keep file sizes optimized for web
- Update HTML links when adding new documents
- Consider creating subdirectories for different document types:
  - `papers/` - Research publications
  - `presentations/` - Conference presentations
  - `certificates/` - Awards and certifications
