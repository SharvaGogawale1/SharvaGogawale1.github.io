# Images Directory

This directory contains image assets for your website.

## Required Files

- **profile-photo.jpg** - Your professional headshot (recommended size: 400x400px or larger, square aspect ratio)

## Recommended Specifications

- **Format**: JPG or PNG
- **Size**: 400x400px minimum (for crisp display on high-resolution screens)
- **Aspect Ratio**: Square (1:1) for best results with the current layout
- **File Size**: Optimize for web (under 500KB recommended)

## Current Placeholder

The website currently uses a placeholder image from placehold.co. Replace `profile-photo.jpg` with your actual professional photo.

## Additional Images

You can add more images here as needed for:
- Research project visuals
- Conference photos  
- Award images
- Other professional imagery

Make sure to update the corresponding HTML `src` attributes when adding new images.
